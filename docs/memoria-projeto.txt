## Escopo e tese

- **ETE** aqui = transferência de tecnologia: lógica de decantação/tratamento de lodos (ETEs) aplicada ao **refino de concentrados minerais** (monazita, bastnasita) e **reciclagem de eletrônicos**. Não é tratamento de esgoto para obtenção de metais.
- **Insumo:** Soluções Ricas de Lixiviação (PLS) de alta concentração (jazidas, concentrados, mineração urbana).
- **Tese:** Quebra de paradigma econômico — ETE reduz CAPEX e torna viáveis jazidas economicamente marginais; escalabilidade modular.

## Estrutura do repositório (caracore-ete)

- **docs/** — Site exposto no GitHub Pages: index.html (portal), laboratorio_campo_largo.html (Ouro 4.0), apostila, artigo, assets (CSS, img, js, pdf). Links relativos (sem base href).
- **library/** — Documentação técnica (P&D): teoria (Kps, cascata), negócios (viabilidade, feedstock), operacional (deploy).
- **minerador_40/** — App desktop Minerador 4.0: centro de controle gamificado (Campo, Lab, Mercado), motor ETE no back-end. Flask + pywebview ou Electron.
- **src/** — core_engine.py; ete_residencial/ (mg/L, Descarte Seguro); ete_mineracao/ (g/L, %, Pureza 99.9%).
- **aulas/** — Módulos em Markdown (INDEX.md, INDEX_JORNADA.md, MODULO_01–05).
- **Raiz** — calculos.py, engine_v2.py, cinetica.py, conformidade.py, analise_economica.py, feedstock_profiles.py; run_residencial.py, run_mineracao.py, run_minerador40.py.
- **scripts/** — build_dist_local.py (gera dist/Minerador40.exe e checksums).
- **electron/** — Shell Electron opcional: cd electron && npm start.
- **CHANGELOG.md**, **library/** (DELIVERY_ETE_RELEASES.md, BLINDAGEM_IP_RELEASE.md, etc.).

## Comandos úteis

- **Minerador 4.0 (desktop):** `python run_minerador40.py` (pywebview). Ou Electron: `cd electron && npm install && npm start`.
- **Instalador Windows:** `python scripts/build_dist_local.py` → dist/Minerador40.exe (recomendado Python 3.10.x para alinhar com CI).
- **ETE Residencial (mg/L):** `python run_residencial.py` ou `streamlit run src/ete_residencial/app_residencial.py`.
- **ETE Mineração (g/L, pureza):** `python run_mineracao.py` ou `streamlit run src/ete_mineracao/app_mineracao.py`.
- **Manual PDF (Ouro 4.0):** `python gerar_manual_ouro40_pdf.py` → docs/assets/pdf/Manual_Iniciacao_Ouro40.pdf.
